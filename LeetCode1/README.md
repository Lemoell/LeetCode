## Sobre o exercício:

Exercício 1 do LeetCode para pegar um vetor e retornar os indices que somando seus valores são iguais ao target

### Retornar os indices da soma

Dado um array de inteiros nums e um inteiro target, retorne os índices dos dois números de forma que a soma deles seja igual a target

### Exemplo

Entrada: nums = [2,7,11,15], target = 9
Saída: [0,1]
Explicação: Como nums[0] + nums[1] == 9, retornamos [0, 1].
